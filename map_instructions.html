Welcome to my mileage calculator project!


Instructions

  • Create a route on the embedded map by clicking on points on the map, or by entering an address in the text fields.
  • Enter the number of times in a given time period you travel along this route, and select the time period from the dropdown menu.
  • Click the green "+ add" button when you are ready to add your route to the table. If the add button is not green, make sure you have a complete route selected and a valid number in the input field.
  • Your entered routes will appear in a scrollable table underneath the add button, and the total annual mileage calculated from these routes will be displayed in the green box below the table.
